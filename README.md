## Weather-App
##### How it works: Enter the name of any recognised city into the input field and submit. Current weather data of the specified city will be displayed on your web page.

* Technologies used:
   * HTML5
   * CSS3
   * JS
   * Openweathermap API
        
* What I Learnt: 
  * Handling APIs
  * Utilising XMLHTTP Requests
  * Handling JSON Datas
